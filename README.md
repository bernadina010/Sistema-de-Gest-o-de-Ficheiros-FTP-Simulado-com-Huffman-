# Sistema-de-Gest-o-de-Ficheiros-FTP-Simulado-com-Huffman-
Um Sistema de Gestão de Ficheiros inspirado no funcionamento de um sistema FTP (File Transfer Protocol), implementado em linguagem C. O sistema permite a gestão de utilizadores, organização de ficheiros em pastas, transferência de ficheiros entre utilizadores e compressão de dados utilizando o algoritmo de Huffman.
